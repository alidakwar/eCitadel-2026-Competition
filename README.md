# eCitadel IV Cyber Defense Competition

## Overview

Participated in the 2026 eCitadel IV National Cyber Defense Competition as the primary defender for the Windows Server and Active Directory environment. The competition required teams to:

- Secure compromised infrastructure
- Maintain critical business services
- Complete operational injects
- Respond to active red-team activity while preserving system functionality

The environment consisted of:

- Windows Server 2022 Domain Controller
- Linux-based production servers
- pfSense firewall infrastructure
- Continuously scored services
- Simulated business operations and incident response requirements

---

## Responsibilities

- Led defense of the Windows Server 2022 and Active Directory environment
- Maintained availability of critical DNS and SSH services throughout the competition
- Investigated and remediated unauthorized accounts, privilege escalation opportunities, persistence mechanisms, and system misconfigurations
- Hardened Active Directory users, groups, permissions, and authentication settings
- Responded to simulated red-team activity through identification, containment, and remediation efforts
- Completed operational injects while preserving production functionality
- Documented findings and incident response actions under competition reporting requirements
