# Competition Overview

## Objective

The eCitadel IV Cyber Defense Competition challenged participants to secure and maintain a simulated enterprise environment while preserving business operations.

Teams were responsible for:

- Securing compromised systems
- Maintaining critical services
- Responding to active adversary activity
- Completing operational business injects
- Investigating and remediating security findings
- Documenting incident response actions

---

## Environment

### Infrastructure

- Windows Server 2022 Domain Controller
- Linux-based Production Servers
- pfSense Firewall

### Critical Services

- DNS
- SSH
- HTTP Web Services
- Ticketing Services

### Security Challenges

The environment contained pre-existing vulnerabilities, unauthorized accounts, privilege escalation paths, persistence mechanisms, and active adversary activity.
